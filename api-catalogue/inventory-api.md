# Inventory API

## Description
API de gestion des stocks et inventaires produits. Suivi en temps réel des niveaux de stock et alertes de rupture.

## Team
Equipe Logistique

## Version
v1

## Endpoints
- GET /v2/inventory
- GET /v1/inventory/{productId}
- PUT /v1/inventory/{productId}/stock
- POST /v1/inventory/alerts
